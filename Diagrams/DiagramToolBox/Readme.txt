The DesignToolBox example allows you to drag & drop various shapes from a custom toolbox control. The example also supports the standard available diagram features such as:
</br>- Drag & Drop of shapes and connections
</br>- Shapes Rotation & Resizing
</br>- Cut/Copy/Paste/Delete/Undo/Redo 
</br>- Pan and Zoom
</br>- Snap to Grid 
</br>- Align Items 
</br>- Graph Layout 
</br>- Read-only Diagrams  
</br>- Optimized performance through Virtualization 
</br>- Save/Load Layout through Serialization 
</br>- Various Mouse tools 
</br>- Rich Commands and Events Model 
</br>- MVVM Support 
This is a strip version of the QSF sample DiagramToolBox.
